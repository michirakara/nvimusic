# Dependencies
- paplay

# Environment
- UNIX/Linux

# Available commands
- Nvimusic
    - Show tui of playlist
- NvimusicAdd
    - Add a folder(does not search recursively) or file to the end of the playlist
- NvimusicShuffle
    - Shuffle the playlist
- NvimusicPlay
    - Start playing
- NvimusicStop
    - Stop playing
- NvimusicSkip
    - Skip to the next music
